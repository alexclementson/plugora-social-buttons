Modern floating social media bar — Facebook, X, WhatsApp, Instagram, LinkedIn, YouTube, TikTok, email, phone and custom links.

== Description ==
Plugora Floating Social Buttons adds a polished, accessible floating social bar to your WordPress site. Drag-to-reorder buttons, official platform brand colours, four positions, multiple layouts (vertical, horizontal, expandable, pop-up), per-page visibility rules and full mobile/desktop control. Lightweight, dependency-free and built to the Plugora standards.

**Free**
* All major platforms — Facebook, X, Instagram, LinkedIn, YouTube, TikTok, WhatsApp, phone, email, SMS, custom
* Drag-to-reorder, enable/disable per button
* Four positions × four layouts
* Device + content-type visibility rules
* Live preview in the admin

**Premium**
* Unlimited custom buttons
* Scheduling / campaign windows
* Advanced page-level visibility
* Priority support

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/
2. Activate from Plugins → Installed Plugins
3. Visit Plugora → Social Buttons to configure
